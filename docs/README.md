## Quick Start

To get started with the Figma MCP server:

1. Install dependencies:
   ```bash
   make install
   ```

2. Configure your Figma API token in `.env`

3. Start the server:
   ```bash
   make mcp
   ```

For more detailed instructions, see the [Usage Guide](./04-usage-guide.md).
